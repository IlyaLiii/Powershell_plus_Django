ТЗ:
  Написать скрипт, который один раз в час будет проверять наличие файлов с расширением *.xml в определенном каталоге.
  Если файл существует, то его надо отправить на веб сервер по протоколу https

1) Создать API для приема XML-файлов на локальный веб-сервер
    Смоделировать API для приема POST-запроса с файлом XML
    Связать View с url

2) Создать скрипт powershell:
    Создать бесконечный цикл для чтения пути с консоли (пока не ввести валидное значение)
    Создать глобальный цикл для отправки запросам каждый час
    Написать проверку наличия файла по заданному пути
    Брать первый файл из списка файлов в директории для дальнейшей отправки на сервер
    Сформировать запрос (Прописать Headers, взять файл из предыдущего пункта)
    Отправить запрос, вернуться к циклу через час

3) Деплой проекта на Github
    Добавить .gitignore
    Создать репозиторий и задеплоить проект
